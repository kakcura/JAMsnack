<h1 align="center">
  What is JAMsnack?
</h1>

A web app to help making healthy eating decisions. Just take a picture of your food and see if it is healty or not.

Happy Healty Snack Jamming!!!

INSTALL
=======

```bash
$ sudo npm install -g gatsby-cli
$ npm install
```

RUN
=======

```bash
$ gatsby develop
```

Navigate
=======

```html
http://localhost:8000
```
